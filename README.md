# Free Workout Timer

An interval timer for Tabata, HIIT, and any work/rest workout you want to run on a phone. No paywall, no ads, no tracking, no accounts.

## What it does

Build a timer once, run it as many times as you want. Set work and rest durations, number of rounds, and any number of sets. Audio cues, haptics, and voice countdowns guide you through the workout. The screen stays awake while a timer is running.

## Who it's for

Anyone who's tired of paywalls, ads, and account walls in workout timer apps. If you've used Tabata Pro, Seconds, or Interval Timer and bounced off the price or the upsells, this is the same shape of tool with none of that.

## How to get it

Coming soon to the App Store and Play Store.

In the meantime, run it locally — see below.

## Run it locally

You'll need [Node.js](https://nodejs.org/) and the Expo Go app on your phone, or an iOS simulator / Android emulator.

```bash
git clone https://github.com/Josh-Approved/workout-timer.git
cd workout-timer
npm install
npm start
```

Then scan the QR code with Expo Go (or press `i` for iOS simulator, `a` for Android emulator).
